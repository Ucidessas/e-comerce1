┃  PASO 1️⃣  - COPIA LOS ARCHIVOS (2 minutos)                            ┃
┃                                                                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  A) Crea una carpeta nueva en tu PC:
     ┌────────────────────────────────────────────┐
     │ Windows:  C:\Proyectos\delivery-local-v3\  │
     │ Mac/Linux: ~/Proyectos/delivery-local-v3/  │
     └────────────────────────────────────────────┘

  B) Copia TODOS estos archivos de /codigo-pc-2/:
     ✅ tailwind.config.js
     ✅ postcss.config.js
     ✅ package.json
     ✅ vite.config.ts
     ✅ tsconfig.json
     ✅ tsconfig.node.json
     ✅ index.html
     ✅ .gitignore
     ✅ instalar.bat (o instalar.sh si usas Mac/Linux)
     ✅ src/ (carpeta completa)
     ✅ styles/ (carpeta completa)

  C) Copia estos archivos de TU proyecto original:
     ⚠️  App.tsx
     ⚠️  /components/ (carpeta COMPLETA con todos los archivos)
     ⚠️  /data/ (carpeta COMPLETA)
     ⚠️  /public/ (opcional, si tienes imágenes)

┃  PASO 2️⃣  - INSTALA LAS DEPENDENCIAS (5 minutos)                      ┃
┃                                                                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  🪟 Si usas WINDOWS:
     ┌────────────────────────────────────────────┐
     │ Doble clic en: instalar.bat                │
     └────────────────────────────────────────────┘
     
     El script automáticamente:
     • Verifica que Node.js esté instalado
     • Limpia instalaciones anteriores
     • Instala todas las dependencias
     • Verifica que todo esté correcto

  🐧 Si usas MAC/LINUX:
     ┌────────────────────────────────────────────┐
     │ chmod +x instalar.sh                       │
     │ ./instalar.sh                              │
     └────────────────────────────────────────────┘

  💻 Si prefieres INSTALACIÓN MANUAL:
     ┌────────────────────────────────────────────┐
     │ npm install                                │
     └────────────────────────────────────────────┘

  ⏱️  Tiempo estimado: 2-5 minutos dependiendo de tu conexión

  │                                            │
